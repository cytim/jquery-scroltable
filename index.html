Quick Start

  1. Install jQuery (1.7+)

  2. Download ScrolTable

  3. Copy ScrolTable images to {{ "/images/" | escape }} folder

  4. Include ScrolTable CSS and JS files into the {{ "" | escape }}

    {% highlight html %} {% endhighlight %} {% highlight html %} {% endhighlight %}

  5. Call the scroltable method to initialize scrollable tables.

    {% highlight js %} $('#sample-table').scroltable(); {% endhighlight %}

Methods

Method Arguments Description
scroltable (none) Initializes ScrolTable
destory (none) Destroys ScrolTable

Example

Destroy the scroltables.

{% highlight js %} $('#sample-table').scroltable('destroy'); {% endhighlight %}
